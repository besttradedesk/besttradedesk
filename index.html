Setup

Live in 5 minutes, literally.

1

Duplicate the Notion Template

Click the link, hit "Duplicate" in Notion. Your database is ready with all views and properties pre-configured.

2

Create a Notion Integration

Go to notion.so/my-integrations, create an internal integration, copy the secret token.

3

Set Up IBKR Flex Query

In IBKR Account Management, create a Flex Query for Options trades. Get your Token + Query ID from Settings → Flex Web Service.

4

Fill in .env and Run

Copy .env.example to .env, paste your 4 credentials, run python ibkr_sync.py. Done. Trades appear in Notion instantly.

FAQ

Common questions

Do I need TWS or IB Gateway running?
No. The script uses IBKR's Flex Query web API, which works with just your browser-based IBKR account. No desktop software required.
Do I need to know Python?
No coding required. You only need to fill in 4 values in a text file (.env) and run one command. The guide walks you through everything step by step.
Does it work with paper trading accounts?
Yes. IBKR Flex Query works for both live and paper trading accounts. Just make sure you create the Flex Query in the correct account.
What option strategies are supported?
All single-leg options: long calls, long puts, short calls (CSP, covered calls), short puts. FIFO matching handles partial fills and multi-day entries. Multi-leg spreads show as individual legs.
How often does it sync?
You control the frequency. Run it manually after trading hours, or schedule it to run automatically every hour/day. The script won't create duplicates no matter how often you run it.
Can I customize the Notion database?
Absolutely. It's your Notion workspace — add, remove, or rename any fields. The sync script uses property names that you can edit in the code.
